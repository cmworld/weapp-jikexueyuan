# 微信小程序－极客学院

### 说明：

实现文章列表，文章详情等功能，特色：
- 实现右侧菜单栏
- 实现显示markdown内容

### 数据接口:

使用本地数据接口

### 目录结构：

- image — 存放项目图片文件
- markdown — 存放markdown解析文件
- page — 存放项目页面渲染相关文件
- util — 存放文章数据

### 开发环境：

微信web开发者工具 v0.11.112301
